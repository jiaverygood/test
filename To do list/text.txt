할 일 목록 앱의 UI만 만들어줘.

이번 단계에서는:
- HTML/CSS만 작성
- JavaScript는 빈 intApp() 함수만
- 기능 구현 없음(추가/삭제/저장없이)
- 할 일 목록은 비워져 있음

파일 구조:
- index.html
- style.css
- main.js

필요한 요소:
- 제목: 나의 할 일 목록
- 입력창(placeholer: "할 일을 입력하세요.")
- 추가 버튼
- 할 일 목록 영역

디자인:
- 중앙 정렬, 최대 너비 500px
- 현대적이고 깔끔한 카드 스타일
- 완료된 항목은 취소선 표시

할 일 추가/삭제/완료 기능을 구현해줘.

이번 단계에서는:
- LocalStorage 저장 기능 없이
- 메모리(배열)에만 저장

구현할 기능:
1. 추가 버튼 클릭 시 할 일 추가
2. 빈 값이면 "할 일을 입력하세요." 알림
3. 체크 박스로 완료/미완료 토글
4. 삭제 버튼으로 항목 제거
