환율 변환기를 만들어줘.

파일 구조:
- index.html
- style.css
- main.js

필요한 요소
금액 입력창(숫자만)
From 통화 선택(드롭다운): USD, EUR, KRW, JPY, SGD, CNY
To 통화 선택(드롭다운)
통화 변환 버튼
결과 표시 영역

주의
위 기능 이외의 다른 기능은 구현하지 않을 것

API: https://open.er-api.com/v6/latest/USD

환율 변환기에 통화 스왑 기능을 추가해줘.

요구사항:
- From과 To 드롭다운 사이에 "🔄 통화 교환" 버튼 추가(ID: swap-btn)
- 버튼 클릭 시 From과 To 통화가 서로 바뀜
- 금액이 입력되어 있으면 교환 후 자동으로 변환 실행
